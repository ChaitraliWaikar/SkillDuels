# SkillDuels 

A real-time quiz battle app where players compete head-to-head or practice solo.

## Current Features (MVP – Working)
- Responsive dashboard with sidebar & navbar
- Home page with stats (XP, Badges)
- Profile page with stats & edit support
- Category selection → Live Battle or Solo Practice
- StartBattle screen with 5-sec countdown
- Quiz interface with timer, scoring, and result navigation
- Solo mode fully supported (no opponent shown)
- Friends system with add/search/requests (mock data)
- BattleResults page with XP, badge, and win/lose display
- Clean, modern teal-themed UI (fully responsive)

## Tech Stack
- React + Vite
- React Router DOM (client-side routing)
- Lucide React (icons)
- Socket.IO client (ready for real-time)
- Pure CSS (no Tailwind – custom responsive design)
